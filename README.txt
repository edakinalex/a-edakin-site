# Сайт-визитка (статические файлы)

## Содержимое
- `index.html` — страница
- `styles.css` — стили
- `script.js` — мелкая логика (год в футере)
- `assets/` — фото и PDF

## Как разместить на домене
Любой хостинг со статическими файлами подойдет (nginx/apache, Timeweb/Reg.ru, GitHub Pages, Netlify, Vercel).

### Вариант 1. Обычный хостинг
1) Откройте панель хостинга → файловый менеджер.
2) Загрузите **весь** каталог (с `index.html`) в корень сайта (обычно `public_html/`).
3) Проверьте, что `index.html` лежит прямо в корне.
4) Откройте домен в браузере.

### Вариант 2. Nginx (пример)
```
server {
  server_name a-edakin.ru www.a-edakin.ru;
  root /var/www/a-edakin.ru;
  index index.html;
}
```

## Примечания
- Кнопка «Связаться» ведет к блоку контактов (якорь `#contacts`).
- Все PDF открываются в новой вкладке (атрибут `target="_blank"`).
- «Полная трудовая биография (PDF)» находится под контактами.
